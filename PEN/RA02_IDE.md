## 1. RA2 Instalación, configuración y uso de cuatro entornos de desarrollo para compilar y ejecutar un programa Java

---

## 2. Objetivo

* Instale y configure **cuatro entornos integrados de desarrollo (IDE)** distintos.
* Escriba un **programa Java con estructuras de control**, sin programación orientada a objetos.
* Compile y ejecute el programa en cada uno de los IDE seleccionados.
* Documente todos los pasos mediante una **memoria técnica en PDF**, incluyendo capturas y comentarios personales.


## 3. Descripción de la tarea

Cada alumno deberá:

1. Elegir **cuatro IDE diferentes** que permitan desarrollar y ejecutar programas en Java.
   
2. Instalar cada uno de los cuatro IDE elegidos y realizar la **configuración inicial necesaria** para poder compilar código Java.

3. Escribir un **programa Java único**, el mismo para los cuatro IDE, que incluya:

   * Lectura de datos.
   * Condicionales (`if` / `else`).
   * Algún bucle (`for`, `while` o `do while`).
   * Cálculos o procesamiento de datos.
     **No debe utilizar POO.** Es un programa de estructura secuencial + selección + repetición.

4. Compilar y ejecutar el programa en cada IDE, asegurándose de que funciona correctamente.

5. Elaborar una **memoria técnica en PDF** que documente todo el proceso.

---

## 4. Contenido obligatorio de la memoria (PDF)

La memoria debe ser breve, clara y estructurada. Debe incluir, para cada uno de los **cuatro IDE**:

**A. Evidencia de instalación**

Una captura que muestre la instalación o la pantalla final del instalador.

**B. Evidencia de configuración inicial**

Una captura donde se vea:

* La selección del SDK o JDK.
* La detección del lenguaje Java o la creación del proyecto.

**C. Evidencia de compilación y ejecución**

Una captura donde se vea el programa ejecutándose dentro del IDE (terminal integrada o consola propia del entorno).

**D. Comentario personal**

Breve reflexión donde el alumno indique:

* Qué le ha gustado más de ese IDE.
* Qué le ha gustado menos.
* Si lo utilizaría de nuevo o no, y por qué.

## 5. Entrega

El alumno deberá subir al Aula Virtual:

* **Un único archivo PDF** con la memoria completa.
